# three-dim-util

A library of utility functions for prototyping 3D graphics applications.

## Dependencies

- OpenGL
- GLUT
- Eigen <http://eigen.tuxfamily.org/>
- Qt <https://www.qt.io/>
- glm <https://glm.g-truc.net/>

## Projects using three-dim-util

- OptiMo <https://github.com/yuki-koyama/optimo>
